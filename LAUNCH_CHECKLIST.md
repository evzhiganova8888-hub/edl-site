- [ ] Открыть Chrome DevTools → Console на каждой странице. Не должно быть красных ошибок (предупреждения от Calendly допустимы)

  - Accessibility ≥ 95
  - Best Practices ≥ 90
  - SEO ≥ 95

- [ ] То же для диагностики — переход на `/diagnostic`
- [ ] Footer ссылки работают (телеграм, email открываются)
- [ ] Поскроллить до конца — header при скролле получает blur-фон

### Mobile

- [ ] Открыть на iPhone (реальный девайс или Chrome DevTools → iPhone 13/14/15)
- [ ] Гамбургер-меню открывается, ссылки кликабельны
